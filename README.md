# 📅 Sistema de Agendamento

Este é um sistema web desenvolvido para a gestão, marcação e controle de agendamentos de horários. A aplicação foi criada para oferecer uma interface intuitiva onde usuários ou clientes podem selecionar datas, horários e serviços disponíveis, além de permitir que administradores gerenciem a agenda de forma centralizada.

## 🚀 Demonstração

O projeto está hospedado e pode ser acessado diretamente pelo navegador através do link:
👉 [Acessar o Sistema de Agendamento](https://kevintav2025.github.io/SistemaAgendamento/)

---

## 💻 Telas Integradas

O sistema é dividido em interfaces funcionais para otimizar a experiência de marcação de horários:

* **Página Inicial (`index.html`):** Painel de entrada com calendário visual, informações de funcionamento e atalhos de navegação.
* **Autenticação (`login.html`):** Área de login para clientes ou administradores acessarem seus painéis.
* **Nova Reserva / Agendamento:** Tela com formulários para seleção de profissional, serviço, data disponível e horários livres.
* **Meus Agendamentos:** Listagem das consultas ou horários marcados pelo usuário, permitindo o histórico e cancelamentos.
* **Painel Administrativo:** Interface focada na gestão de vagas, bloqueio de datas, configuração de horários e relatórios de atendimento.

*(Nota: Caso os arquivos HTML no seu VS Code tenham nomes específicos como `telaCalendario.html` ou `agenda.html`, lembre-se de ajustar esta lista diretamente no arquivo).*

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação dos formulários de agendamento, seletores de data/hora e tabelas.
* **CSS3:** Layout moderno, limpo e responsivo para garantir que os agendamentos possam ser feitos pelo celular ou computador.
* **C# / JavaScript:** Lógica de programação para validação de campos, bloqueio de horários duplicados na agenda e manipulação de datas.
